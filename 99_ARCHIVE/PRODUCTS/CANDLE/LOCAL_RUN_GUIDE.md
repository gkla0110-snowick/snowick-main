# SNOWICK 캔들 로컬 실행 가이드

## 목적
캔들 5향 등록 파일을 PC에서 생성하고 확인할 때 사용하는 기준.

## 사용 파일

```text
seo_generate.py
SNOWICK_Cafe24_등록시트_5향_대나무확정.md
```

## 인코딩 기준

```text
UTF-8
BOM 없음
메모장 저장 금지
```

## 실행 순서

```text
1. 이미지 폴더 정리
2. seo_generate.py 위치 확인
3. 터미널 실행
4. 등록시트 생성 확인
5. Cafe24 등록값 복사
6. Cafe24 수동 등록
7. 네이버 스마트스토어 등록
8. 쿠팡 등록
```

## 완료 기준

```text
캔들 5향 Cafe24 + 네이버 + 쿠팡 등록 완료
```
