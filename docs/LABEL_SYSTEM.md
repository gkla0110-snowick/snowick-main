────────────────────────
스노윅 라벨 방향
────────────────────────

스노윅 라벨은 감성보다 구조와 재현성을 우선한다.

출력 결과는:

1. 안정적이어야 하며
2. 반복 가능해야 하고
3. 브랜드 구조를 유지해야 한다.
4. 제품군 기준을 유지해야 한다.
5. 출력 품질이 일정해야 한다.

제품명만 변경되어도 전체 구조는 유지 가능해야 한다.
