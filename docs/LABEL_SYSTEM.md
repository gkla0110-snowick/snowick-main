# SNOWICK LABEL SYSTEM FULL MASTER

---

## 시스템 목적

스노윅 라벨 시스템을:
재현 가능한 출력 구조 기준으로 통합 관리한다.

라벨은:
단순 디자인이 아니라

- 브랜드 구조 유지
- 출력 안정성 유지
- 반복 생산 가능성 유지
- 제품군 일관성 유지

를 목적으로 운영한다.

---

## 핵심 운영 방향

- 규격 고정
- PNG 우선 출력
- 로고 절대 고정
- 제품명 치환 구조 유지
- 반복 가능한 출력 유지
- 감성보다 구조 우선

---

## 기본 운영 기준

- 파일명 영문 고정
- snake 및 underscore 구조 유지
- 내부 문서는 한글 사용 가능
- 로고 임의 생성 금지
- 첨부 로고만 사용
- 규격 임의 변경 금지

# 출력 구조 기준

---

## PNG 운영 기준

- PNG를 최종 출력 기준으로 사용한다.
- 인쇄 기준 가독성을 우선한다.
- 모바일 확인 기준도 함께 유지한다.
- 반복 출력 시 품질 편차를 최소화한다.

---

## SVG 운영 기준

- SVG는 마스터 구조 유지용으로 사용한다.
- 좌표 구조 임의 변경 금지
- 로고 위치 변경 금지
- 정렬 구조 유지
- 텍스트 구조 유지

# 스노윅 라벨 방향

스노윅 라벨은:
감성보다 구조와 재현성을 우선한다.

출력 결과는:

- 안정적이어야 하며
- 반복 가능해야 하고
- 브랜드 구조를 유지해야 한다.

제품명만 바뀌어도
전체 구조는 유지 가능해야 한다.
