# SNOWICK CONTENT SYSTEM FULL MASTER

---

## 시스템 목적

스노윅 콘텐츠 구조를:
브랜드 기준 형태로 통합 관리한다.

콘텐츠는:
단순 감성 표현이 아니라

- 브랜드 구조 전달
- 차별점 전달
- 전환 유도
- 운영 일관성 유지

를 목적으로 운영한다.

---

## 핵심 운영 방향

- 구조 우선
- 감성 과부하 금지

- 플랫폼별 구조 분리
- 재현 가능한 콘텐츠 유지

2. 제품군 판별
3. 핵심 구조 선정
4. 차별점 선정
5. 전환 흐름 구성
6. 플랫폼 정책 검토
7. 최종 검수

# 실제 운영 데이터 누적 기준

CONTENT_SYSTEM 내부에는:

- 반응 좋았던 콘텐츠
- 조회수 높았던 구조
- 전환 높았던 문장
- 실패 콘텐츠 사례
- 플랫폼 정책 대응 사례

를 누적 기록한다.

# 스노윅 CONTENT 방향

스노윅 콘텐츠는:
감성보다 구조를 우선한다.

콘텐츠는:

- 브랜드 기준
- 구조 차별성
- 안정성
- 재현성

을 전달할 수 있어야 한다.
