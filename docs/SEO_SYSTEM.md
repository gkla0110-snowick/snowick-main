# 스노윅 SEO 시스템

## 핵심 규칙

- 구조 우선
- 감성적 과부하 없음
- 검색 최적화
- 재현 가능한 형식
- 플랫폼 분리

---

## 플랫폼

- 카페24
- 네이버 스마트스토어
- 쿠팡
- 구글 SEO

---

## 제품 규칙

- 감성보다 검색 구조 우선
- 제품 용도 우선 표기
- 검색형 키워드 우선 사용
- 플랫폼별 구조 분리 유지
- 검증된 키워드만 사용
- 브랜드 남용 금지
- 중복 키워드 반복 금지

---

## 카페24 규칙

- 상품요약설명 40자 이내
- 간략설명 40자 이내
- 검색어 10개 고정
- 검색어는 콤마 구분
- 메타태그1 = 스노윅 고정
- 메타태그2 = SEO 설명
- 메타태그3 = 검색어 복붙 금지
- ALT = 대표 이미지 제품명 사용
- 감성 문구보다 검색 구조 우선

---

## 스토어팜 규칙

- 자동완성 기반 키워드 우선
- 태그 10개 유지
- 플랫폼 제한어 검증
- 검색형 키워드 우선
- 감성 과부하 금지
- 제품 용도 우선 표기

---

## 쿠팡 규칙

- 태그 20개 가능
- 중복 키워드 금지
- 특수문자 최소화
- 검색 구조 우선
- 제품 용도 우선 작성

---

## 구글 SEO 규칙

- Page Title 구조 유지
- ALT와 메타 역할 분리
- 제품명 중복 최소화
- 향조 기반 메타 설명 작성
- 검색 의도 우선 구조 유지

---

## ALT 규칙

- 대표 이미지 = 제품명
- 원료 이미지 = 향료원액
- 용량 이미지 = 제품명 + 용량
- 연출 이미지 = 제품 용도 기반 작성

---

## 메타 규칙

- 메타와 검색어 동일 복붙 금지
- 감성 문구보다 구조 설명 우선
- 향조 기반 설명 사용
- 플랫폼 제한 글자수 우선 준수

---

## 금지 사항

- 플랫폼 동일 문장 복붙
- 과도한 감성 문구
- 검증 안된 키워드 사용
- SEO와 상세페이지 혼합
- 퍼퓸 제품 화장품 오인 표현
- 특수문자 남용
- 구조 없는 감성 제목 사용

---

## 스노윅 SEO 방향

스노윅은 감성보다 구조를 우선한다.

검색 유입 이후:
- 구조
- 안정성
- 재현성
- 설계 기준

중심으로 전환을 유도한다.
