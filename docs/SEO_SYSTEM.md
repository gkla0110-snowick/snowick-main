# 스노윅 SEO 시스템

---

## 핵심 규칙

- 구조 우선
- 감성적 과부하 금지
- 검색 구조 우선
- 재현 가능한 형식 유지
- 플랫폼별 SEO 분리 운영
- 검색 의도 중심 구조 유지

---

## SEO 운영 목적

SEO는 단순 노출 목적이 아니다.

스노윅 SEO는:

- 검색 유입
- 구조 전달
- 차별점 전달
- 전환 유도

를 동시에 수행해야 한다.

---

## 플랫폼 구조

- 카페24
- 네이버 스마트스토어
- 쿠팡
- 구글 SEO

플랫폼별 규칙은 반드시 분리 운영한다.

---

## 공통 SEO 기준

- 제품 용도 우선 표기
- 감성보다 검색 구조 우선
- 검증된 키워드만 사용
- 플랫폼별 문장 분리 유지
- 브랜드명 남용 금지
- 중복 키워드 반복 금지
- ALT / 메타 / 검색어 역할 분리
- 플랫폼 정책 우선 준수

---

## 카페24 SEO 규칙

- 상품요약설명 40자 이내
- 간략설명 40자 이내
- 검색어 10개 고정
- 검색어는 콤마 구분
- 메타태그1 = 스노윅 고정
- 메타태그2 = SEO 설명
- 메타태그3 = 검색어 복붙 금지
- ALT = 대표 이미지 제품명 사용
- 특수문자 최소화
- 제품군별 구조 유지

---

## 스마트스토어 SEO 규칙

- 자동완성 기반 키워드 우선
- 태그 10개 유지
- 홍보문구 50BYTE 이하 유지
- 플랫폼 제한어 검토 후 사용
- 감성 과부하 금지
- 검색형 키워드 우선
- 제품 용도 중심 설명 유지

---

## 쿠팡 SEO 규칙

- 태그 20개 가능
- 검색형 키워드 우선
- 중복 키워드 최소화
- 제품 용도 우선 작성
- 특수문자 최소화
- 플랫폼 검색 구조 우선

---

## 구글 SEO 규칙

- Page Title 구조 유지
- ALT와 메타 역할 분리
- 제품명 반복 최소화
- 향조 기반 메타 설명 작성
- 검색 의도 중심 구조 유지
- 플랫폼별 문장 복붙 금지

---

## ALT 규칙

- 대표 이미지 = 제품명
- 원료 이미지 = 향료원액
- 용량 이미지 = 제품명 + 용량
- 연출 이미지 = 제품 용도 기반 작성
