## 2. seo_generate.py 입력 포맷

### 완제품
| 타입 | 포맷 | 예시 |
|---|---|---|
| 캔들 | 제품명/캔들/용량/단가(선택) | 시그니처대나무캔들/캔들/200g |
| 퍼퓸 | 제품명/퍼퓸/단가(선택) | 한라봉/퍼퓸 |
| 디퓨저 | 제품명/디퓨저/용량/단가(선택) | 비자림/디퓨저/100ml |
| 스타윅 | 스타윅/지름mm/단가(선택) | 스타윅/60mm |
| 오일 | 제품명/오일/용량/단가(선택) | 매화꽃/오일/100ml |

### 부자재 — 고정 규격
| 품목키 | 허용 규격 |
|---|---|
| 왁스_실크소이 | 1kg |
| 왁스_비즈화이트 | 1kg |
| 왁스_비즈옐로우 | 1kg |
| 스톤_샤쉐 | 500g / 1kg |
| 스톤_레진 | 500g / 1kg |
| 베이스_디퓨저 | 500ml / 1L |
| 베이스_오드뚜왈렛 | 500ml / 1L |
| 베이스_섬유탈취 | 500ml / 1L |
| 스틱_블랙 | 3mm*30cm / 4mm*30cm / 4mm*40cm / 5mm*60cm |
| 스틱_화이트 | 3mm*30cm / 4mm*30cm / 4mm*40cm / 5mm*60cm |
| 스틱_브라운 | 3mm*30cm / 4mm*30cm / 4mm*40cm / 5mm*60cm |

입력 포맷: `품목키/규격/단가(선택)`
예) `왁스_실크소이/1kg` / `스틱_블랙/4mm*30cm/500`

### 부자재 — 노멀 (자유 입력)
| 품목키 | 포맷 |
|---|---|
| 캔들용기 | 캔들용기/제품명/용량/단가(선택) |
| 디퓨저용기 | 디퓨저용기/제품명/용량/단가(선택) |
| 퍼퓸용기 | 퍼퓸용기/제품명/용량/단가(선택) |

예) `캔들용기/원형유리용기/200ml/3500`

## 3. 단가 처리 규칙

- 단가는 입력값으로만 반영. AI 임의 작성 금지.
- 단가 입력 시: 출력 헤더에만 표시 (15개 필드에는 미포함)
- 단가 미입력 시: 헤더에 단가 행 자체 미출력

---

## 4. 출력 필드 15개

1. 제품명 Cafe24 (50자 이내)
2. 영문 상품명
3. 상품명 관리용 (50자 이내)
4. 공급사 상품명 — 고정값: 스노윅
5. 상품 요약설명 (255자 이내)
6. 상품 간략설명
7. 검색어 — Cafe24 20개 / 쿠팡 20개 / 네이버 10개
8. 네이버 쇼핑 추가 홍보문구 (50자 이내)
9. 다음 쇼핑하우 추가 홍보문구 (200자 이내)
10. 브라우저 타이틀
11. 메타태그 Author — 고정값: 스노윅
12. 메타태그 Description (80~150자 자동 보정)
13. 메타태그 Keywords (10개)
14. ALT 텍스트 3종
15. 상세페이지 6단계 카피
