# SNOWICK SHORTFORM SYSTEM FULL MASTER

PROJECT:
SNOWICK_SHORTFORM_SYSTEM_FULL_MASTER_V1

────────────────────────
시스템 목적
────────────────────────

스노윅 숏폼 콘텐츠 구조를 브랜드 기준으로 통합 관리한다.

숏폼은 단순 영상 업로드가 아니라 다음 목적을 위해 운영한다.

1. 브랜드 구조 전달
2. 차별점 시각화
3. 신규 유입 유도
4. 전환 흐름 연결
5. 플랫폼 알고리즘 대응
6. 재현 가능한 콘텐츠 유지
7. 브랜드 일관성 유지

────────────────────────
핵심 운영 방향
────────────────────────

운영 기준:

1. 첫 1초 시선 정지 우선
2. 구조 차별점 우선 전달
3. 감성 과부하 금지
4. 플랫폼별 구조 분리
5. 브랜드 기준 유지
6. 재현 가능한 포맷 유지
7. 자막 구조 기반 유지

────────────────────────
숏폼 핵심 전달 요소
────────────────────────

우선 전달 요소:

1. 특허심지 구조
2. 별모양심지 시각화
3. 멜트풀 형성 구조
4. 터널링 0% 설계 근거
5. 안정적 불꽃 구조
6. 공간 중심 향 흐름
7. 구조 기반 재현성

전달 방식:

1. 영상 중심
2. 자막 보조
3. 사운드 강조 (불꽃 소리, 왁스 녹는 소리)
4. 클로즈업 중심
5. 비교 구조 활용

플랫폼별 숏폼 구조
────────────────────────

YouTube Shorts

운영 기준:

1. 첫 컷 — 불꽃 점화 또는 멜트풀 클로즈업
2. 3초 이내 핵심 구조 전달
3. 자막 — 짧고 굵은 구조 문장
4. 특허심지 강조 필수
5. 반복 가능한 포맷 유지
6. 영상 길이 — 30초~60초 기준

금지:

1. 첫 컷 로고 또는 브랜드 소개 금지
2. 과도한 감성 BGM 금지
3. 자막 과부하 금지
4. 영상 길이 과도한 확장 금지

Instagram Reels

운영 기준:

1. 첫 프레임 — 시선 정지 이미지

3. 짧은 자막 구조
4. 브랜드 일관성 유지
5. 피드 연속성 고려
6. 영상 길이 — 15초~30초 기준

금지:

1. 피드 톤과 충돌하는 영상 금지
2. 과도한 편집 효과 금지
3. 브랜드 기준 이탈 금지

TikTok / 기타 플랫폼

운영 기준:

1. 플랫폼 알고리즘 반영
2. 사운드 중심 구조 유지
3. 트렌드 음원 검토 후 활용

────────────────────────
숏폼 포맷 구조
────────────────────────

기본 포맷 (캔들):

컷 1 — 불꽃 점화 클로즈업 (0~3초)
컷 2 — 멜트풀 형성 과정 (3~10초)
컷 3 — 별모양심지 구조 강조 (10~20초)
컷 4 — 공간 연출 (20~30초)
컷 5 — 브랜드 마무리 (30초~)

기본 포맷 (디퓨저):

컷 1 — 디퓨저 스틱 세팅 클로즈업 (0~3초)
컷 2 — 공간 향 흐름 연출 (3~15초)
컷 3 — 잔향 지속 구조 전달 (15~25초)
컷 4 — 브랜드 마무리 (25초~)

자막 구조 기준
────────────────────────

자막 운영 기준:

1. 한 화면당 1~2문장 이내
2. 핵심 키워드 강조
3. 구조 중심 문장 사용
4. 과도한 감성 문장 금지
5. 허위·과장 표현 금지
6. 플랫폼 정책 위반 문구 금지

자막 예시 (캔들):

구조가 터널링을 막습니다.
별모양 특허심지 — 멜트풀이 다릅니다.
타지 않습니다. 구조가 다릅니다.
