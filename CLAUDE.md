## 브랜드 정체성

모든 콘텐츠, 상품 설명, SEO, 캡션은 다음 우선순위를 따릅니다.

```
구조 > 재현성 > 안전성 > 플랫폼 정확성 > 감성 표현
```

```

```

플랫폼별 상품명, SEO, 설명 문장, 태그는 반드시 분리 운영합니다.
카페24 원장 문장을 다른 플랫폼에 그대로 복붙하지 않습니다.

## 플랫폼별 핵심 규칙

### 카페24
- 상품명에 `|` 파이프 문자 사용 금지
- 가격·원산지·제조사 임의 생성 금지
- 상품명 / 브라우저 제목 / 메타 키워드 / 홍보문구 일관성 유지

### 네이버 스마트스토어
- 확정 검색어와 후보 검색어 반드시 분리
- 확정 검색어: 소이캔들, 생일선물, 우드심지, 집들이선물
- 후보 검색어는 판매자센터 입력 확인 전까지 확정 표기 금지
- 검색어 10개 이내

### 쿠팡
- 검색어 최대 20개
- 향 키워드 + 캔들 카테고리 키워드 + 구조 키워드 + 소재 키워드 + 용도·선물 키워드 포함
- 검증되지 않은 기능성 표현 금지

### 인스타그램 / 스레드
- 공통 본문 + 향별 1줄 + 프로필 링크 + 해시태그 구조 유지
- 해시태그 5개 이내
- 과도한 감성 표현 금지

### 숏폼 (YouTube Shorts / Reels)
- 첫 1초: 불꽃 점화 또는 멜트풀 클로즈업
- 자막: 한 화면 1~2문장 이내
- 허위·과장 표현 금지
- 상세 기준: docs/SHORTFORM_SYSTEM.md 참고
