# SNOWICK 캔들 이미지 폴더 규칙

## 목적
캔들 5향 등록 전 PC 이미지 파일을 누락 없이 정리하기 위한 폴더 기준.

## 기본 구조

```text
products/
├─ bijarim/
├─ camellia/
├─ maehwa/
├─ azalea/
└─ hallabong/
```

## 향별 파일 구성

각 향 폴더에는 6개 파일을 둔다.

```text
main_향명.png
detail01.png
detail02.png
detail03.png
detail04.png
detail05_auth.png
```

## 향별 폴더명

| 향명 | 폴더명 | 메인 파일명 |
|---|---|---|
| 비자림숲속 | bijarim | main_bijarim.png |
| 동백꽃 | camellia | main_camellia.png |
| 매화꽃 | maehwa | main_maehwa.png |
| 철쭉꽃 | azalea | main_azalea.png |
| 한라봉 | hallabong | main_hallabong.png |

## 공용 이미지

아래 5장은 전 향 동일하게 복사한다.

```text
detail01.png
detail02.png
detail03.png
detail04.png
detail05_auth.png
```

## 총 파일 수

| 구분 | 수량 |
|---|---:|
| 메인 이미지 | 5장 |
| 공용 상세 이미지 | 20장 |
| 인증 이미지 | 5장 |
| 총합 | 30장 |

## 체크리스트

```text
[ ] 향별 폴더 5개 생성
[ ] 메인 이미지 5장 확인
[ ] 공용 상세 4장 × 5향 복사
[ ] 인증 이미지 1장 × 5향 복사
[ ] 파일명 영문 소문자 유지
[ ] 등록 전 이미지 열어서 제품 컷 확인
```
