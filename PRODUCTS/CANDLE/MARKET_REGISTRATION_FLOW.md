# SNOWICK 캔들 마켓 등록 순서

## 목적
캔들 5향 등록 순서를 고정해 재작업을 막기 위한 실행 문서.

## 등록 순서

```text
1. Cafe24
2. 네이버 스마트스토어
3. 쿠팡
```

## 1단계 Cafe24

| 항목 | 기준 |
|---|---|
| 등록 대상 | 캔들 5향 |
| 등록 방식 | 수동 등록 |
| 가격 | 35,000원 |
| 상품명 | 확정 상품명 패턴 사용 |
| 이미지 | 향당 6장 |
| Author | 스노윅 |

## 2단계 네이버 스마트스토어

Cafe24 등록 완료 후 진행한다.
Cafe24에 입력한 상품명, 가격, 이미지, 검색어 기준을 그대로 맞춘다.

## 3단계 쿠팡

네이버 등록 완료 후 진행한다.
검색어 20개 기준을 별도로 확인한다.

## 완료 체크

```text
[ ] Cafe24 5향 등록 완료
[ ] 네이버 스마트스토어 5향 등록 완료
[ ] 쿠팡 5향 등록 완료
[ ] 이미지 누락 없음
[ ] 가격 오입력 없음
[ ] 상품명 오탈자 없음
```
