# SNOWICK 캔들 완료 후 다음 상품군 자동화 계획

## 목적
캔들 5향 등록 완료 후 다음 상품군 등록 자동화로 넘어가기 위한 기준 문서.

## 착수 조건

```text
캔들 5향 Cafe24 + 네이버 + 쿠팡 등록 완료
```

## 적용 대상

```text
디퓨저
퍼퓸
오일
부자재
```

## 준비 기준

```text
이미지 경로 입력 기준 확정
상품군 판별 기준 확정
15필드 생성 기준 확정
파일 저장 기준 확정
테스트 상품 1개 검증
```

## 작업 순서

```text
1. 캔들 등록 완료 확인
2. 디퓨저 상품군 기준 문서 생성
3. 이미지 기반 등록값 생성 구조 설계
4. 테스트 상품 1개로 검증
5. 문제 없을 때 상품군 확장
```
